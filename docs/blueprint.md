# **App Name**: GymConnect

## Core Features:

- Tab Navigation: Implement a bottom tab navigator with Home, Routine, Progress, and Profile tabs for easy navigation.
- Reusable Components: Create reusable components like PrimaryButton, Card, and FormField for a consistent UI.
- Stack Navigation: Set up stack navigation to manage screens outside the tab bar, such as Onboarding and Exercise Detail, with authentication check.
- Login/Onboarding Screen: Implement a screen to allow a user to log in or create a new account. Placeholders provided for logo, email and password. Session starts after completing form or create flow.
- Home/Dashboard Screen: Render mock workout routine, user streak, and weight progress with pressable cards and chart integration for quick stats access.
- My Routine Screen: Display today's workout list, navigate to exercise details on card press, and integrate circular checkboxes for tracking.
- Progress Screen: Show progress using a react-native-chart-kit, alongside FAB button (functional later).

## Style Guidelines:

- Primary color: A vibrant fitness-oriented blue (#29ABE2) to represent energy and motivation.
- Background color: A dark charcoal (#121212) to provide contrast and focus on the content.
- Accent color: An analogous cyan (#29E2D2) to complement the primary and indicate interactive elements.
- Body and headline font: 'PT Sans' (sans-serif) offers a modern yet warm aesthetic, suitable for both headings and content.
- Use sharp, minimalist icons that visually represent workout routines and fitness goals.
- Employ a clear, vertical layout for dashboard and lists to ensure effortless navigation and readability.
- Incorporate subtle animations, like a slide-in effect for cards and tab transitions, to create a polished and engaging user experience.
